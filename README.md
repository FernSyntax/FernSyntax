## 🚀 The Micro-SaaS Portfolio

Here is a collection of 30 micro-SaaS products, developer tools, and infrastructure services I have launched and am currently building. I focus on creating open-source and self-hosted alternatives to expensive enterprise tools.

### 🌱 Junior Builds (1–10)
* **[Snippr](#)** — A developer-first pastebin with syntax highlighting, expiry, password protection, and an API. Dead simple, incredibly fast. 
    * *Stack:* Go, Gin, Redis, Postgres
* **[Waitlisted](#)** — An embeddable waitlist widget. Drop a `<script>` tag on any landing page to collect emails, show queue positions, and trigger automated alerts. Replaces Waitlist.so.
    * *Stack:* PHP, Laravel, Alpine.js
* **[Patchnotes](#)** — A public changelog widget. Embed a markdown-powered release history popover with auto-generated RSS feeds directly into your product. Replaces Beamer/Headway.
    * *Stack:* SvelteKit, Postgres
* **[Formbase](#)** — An open-source Typeform alternative featuring multi-step forms, conditional logic, file uploads, webhooks, and CSV exports.
    * *Stack:* Ruby on Rails, Hotwire
* **[Crontab.email](#)** — Dead simple heartbeat monitoring for developers. Your cron hits a URL every run; you get an SMTP alert if it fails. Replaces Cronitor.
    * *Stack:* Go, Postgres, SMTP
* **[Invoiced](#)** — A frictionless one-page invoicing tool for freelancers. Create, track, and export PDFs via WeasyPrint, with no recipient account required. Replaces FreshBooks/Wave.
    * *Stack:* C#, ASP.NET Core, WeasyPrint
* **[Replybox](#)** — A privacy-first embeddable comment widget for static sites and blogs. No ads, no tracking, full Markdown support. A drop-in Disqus replacement.
    * *Stack:* PHP, Laravel, Livewire
* **[Openstatus](#)** — A status page builder. Connect uptime checks, publish a public status page at your domain, and alert subscribers during incidents.
    * *Stack:* Elixir, Phoenix, Postgres
* **[Shortstack](#)** — A robust URL shortener with a real-time analytics dashboard featuring click heatmaps, referrer breakdowns, geo-data, and QR generation.
    * *Stack:* Kotlin, Ktor, ClickHouse
* **[Permify](#)** — A role-based access control (RBAC) library and service. Define roles, permissions, and resource scopes in YAML to drop in front of any API. 
    * *Stack:* Go, Postgres, gRPC
